## Overview

Personal dotfiles for arch linux (Gnome and KDE and a bit of windows).

## Installation

Run the `./setup.sh` script

```bash
git clone https://github.com/AndreasRoither/dotfiles && cd dotfiles && bash ./setup.sh
````
